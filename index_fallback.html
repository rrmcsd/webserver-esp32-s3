User Manual

"There is a usable button on the top of your mini TV. Here's how to use it."

Double Click:

Use to navigate through main menu items, to change currencies, to set brightness and to turn the webserver on or off.

Long Click:

Use to enter and exit main menu items.

For better usability, wait 1 second before performing successive clicks. The long click is only activated after 1 second of pressing to.

WiFi Config

*Only 2.4Ghz networks are accepted. Check github.com/GeekMagicClock/GeekMagic-S3 for more information.

Connect to Wi-Fi for more information. After signing up, connect to the same network as applied and access the ⚙️ icon in the item menu to have fun.
